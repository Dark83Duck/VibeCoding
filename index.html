💡 자료 활용 가이드

6 STEPS
01
수업 전 — 커리큘럼 표 확인
FILE 01 커리큘럼 표를 열어 전체 흐름과 해당 차시의 도구·결과물을 파악합니다. 학생·학부모에게 안내장 대신 공유할 수도 있습니다.
02
수업 전 — 계획안 숙지
FILE 02 세부 계획안에서 해당 차시의 5단계 흐름, 교사 멘트, 운영 팁을 미리 읽어 두면 실수 없이 진행할 수 있습니다.
03
수업 전 — 활동지 인쇄
FILE 03 학생 활동지를 Word에서 열어 해당 차시 페이지만 선택 인쇄하거나 PDF로 변환해 배부합니다.
04
수업 중 — 슬라이드 제시
FILE 04 수업 슬라이드를 PowerPoint 또는 Google 슬라이드로 열어 빔 프로젝터에 표시합니다. 차시 표지 슬라이드로 빠르게 이동하세요.
05
수업 중 — 활동지 병행
슬라이드의 미션 슬라이드를 화면에 띄운 채로 학생들이 활동지에 직접 기록하게 합니다. 성찰 질문은 마무리 슬라이드와 연계됩니다.
06
수업 후 — 계획안 갱신
실제 수업 후 계획안의 운영 팁에 현장 메모를 추가해 두면 다음 해에도 개선된 버전으로 재활용할 수 있습니다.
